# Critical Thinking AI Assessment

## Prerequisites

Install **Python 3.10+** (check "Add Python to PATH" during installation)

Download: https://www.python.org/downloads/

## Quick Start

Double-click `start.bat` (or `start.vbs` for silent mode)

## Manual Start

```bash
pip install streamlit
python -m streamlit run app.py
```

Browser opens automatically at http://localhost:8501

## Project Structure

```
app.py                  # Main program (single file)
cases/                  # 5 assessment scenarios
rubrics/rubric.json     # Scoring criteria
start.bat               # Double-click to start
start.vbs               # Silent start (no CMD window)
README.md               # This file
```
